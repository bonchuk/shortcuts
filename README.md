# Порізати відео — шорткат для iPhone 🎬

Ріже будь-яке відео з галереї на частини по N секунд (типово — по хвилині) і зберігає їх назад у «Фото» в правильному порядку. Все відбувається локально на телефоні: без інтернету, без сторонніх сервісів, без перекодування — тому швидко і зовсім без втрати якості.

Зручно для SMM: довге відео → готові шматки для сторіз чи рілсів за кілька секунд.

## Встановлення (один раз, ~10 хвилин)

**1. Встанови a-Shell** — безкоштовний термінал з App Store ([посилання](https://apps.apple.com/app/a-shell/id1473805438)). Потрібна саме повна версія, **не** «a-Shell mini». Відкрий його раз і перевір, що ffmpeg на місці:

```
ffmpeg -version
```

**2. Встанови шорткат** — завантаж файл [`Порізати відео 2.7.shortcut`](./Порізати%20відео%202.7.shortcut) з цього репозиторію прямо на iPhone і відкрий його — він додасться в застосунок «Команди».

**3. Дай a-Shell доступ до папки «Команд»** — в a-Shell набери:

```
pickFolder
```

У вікні вибору перейди: **Огляд → iCloud Drive → Shortcuts** і підтверди. Потім подивись назву створеної закладки:

```
showmarks
```

і перейменуй її на `Shortcuts` (встав точну назву зі списку, зазвичай це `iCloud~is~workflow~my~workflows`):

```
renamemark iCloud~is~workflow~my~workflows Shortcuts
```

Регістр важливий — має вийти саме `Shortcuts` з великої «S».

**4. Перший запуск** — відкрий будь-яке коротке відео в «Фото» → «Поділитися» → «Порізати відео 2.7». iOS попросить кілька дозволів (Фото, запуск команд a-Shell) — скрізь дозволь.

## Використання

1. «Фото» → відео → кнопка **«Поділитися»** → **«Порізати відео 2.7»**.
2. Вкажи довжину частини в секундах (типово 60).
3. Дочекайся сповіщення «Готово!» — частини вже в галереї, по порядку.

## Часті питання

**Чому частини не рівно по N секунд (наприклад, 9–11 замість 10)?**
Різання йде без перекодування — по ключових кадрах відео (раз на 1–2 с). Зате це миттєво і без жодної втрати якості.

**Чому спочатку довго пише «Preparing…»?**
Це iOS готує файл відео; якщо увімкнена «Оптимізація сховища iPhone», оригінал спершу докачується з iCloud.

**Чи потрібен інтернет?**
Ні. Робоча папка має суфікс `.nosync`, тому навіть проміжні файли не синхронізуються з iCloud.

**Помилка `cd: … no such file or directory`?**
Злетіла або неправильно названа закладка — повтори крок 3.

## Як це працює

«Команди» зберігають відео у папку `iCloud Drive/Shortcuts/VideoSplit.nosync` → ffmpeg (усередині a-Shell) ріже його на `part_000.mov`, `part_001.mov`, … потоковим копіюванням (`-c copy -f segment`) → шорткат зберігає частини в галерею по черзі й чистить папку. Технічні деталі та схема всіх дій — у [video-splitter.md](./video-splitter.md).
