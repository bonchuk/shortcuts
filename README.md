## ✂️ Порізати відео

Ріже будь-яке відео з галереї на частини по N секунд (типово — по хвилині) і зберігає їх назад у «Фото» в правильному порядку. Все відбувається локально на телефоні: без інтернету, без сторонніх сервісів, без перекодування — тому швидко і зовсім без втрати якості.

Зручно для SMM: довге відео → готові шматки для сторіз чи рілсів за кілька секунд.

📱 **Візуальна інструкція:** https://claude.ai/code/artifact/b3e3b1dd-1f7e-4367-a5bd-404f142cf949

### Встановлення (один раз, ~10 хвилин)

**1. Встанови a-Shell** — безкоштовний термінал з App Store ([посилання](https://apps.apple.com/app/a-shell/id1473805438)). Потрібна саме повна версія, **не** «a-Shell mini». Відкрий його раз і перевір, що ffmpeg на місці:

```
ffmpeg -version
```

**2. Встанови шорткат** — завантаж файл [`Порізати відео 2.6.shortcut`](./Порізати%20відео%202.6.shortcut) з цього репозиторію прямо на iPhone і відкрий його — він додасться в застосунок «Команди».

**3. Дай a-Shell доступ до папки «Команд»** — в a-Shell набери:

```
pickFolder
```

У вікні вибору перейди: **Огляд → iCloud Drive → Shortcuts** і підтверди. Потім подивись назву створеної закладки:

```
showmarks
```

і перейменуй її на `Shortcuts` (встав точну назву зі списку, зазвичай це `iCloud~is~workflow~my~workflows`):

```
renamemark iCloud~is~workflow~my~workflows Shortcuts
```

Регістр важливий — має вийти саме `Shortcuts` з великої «S».

**4. Перший запуск** — відкрий будь-яке коротке відео в «Фото» → «Поділитися» → «Порізати відео 2.6». iOS попросить кілька дозволів (Фото, запуск команд a-Shell) — скрізь дозволь.

### Використання

1. «Фото» → відео → кнопка **«Поділитися»** → **«Порізати відео 2.6»**.
2. Вкажи довжину частини в секундах (типово 60).
3. Дочекайся сповіщення «Готово!» — частини вже в галереї, по порядку.

### Часті питання

**Чому частини не рівно по N секунд (наприклад, 9–11 замість 10)?**
Різання йде без перекодування — по ключових кадрах відео (раз на 1–2 с). Зате це миттєво і без жодної втрати якості.

**Чому спочатку довго пише «Preparing…»?**
Це iOS готує файл відео; якщо увімкнена «Оптимізація сховища iPhone», оригінал спершу докачується з iCloud.

**Чи потрібен інтернет?**
Ні. Робоча папка має суфікс `.nosync`, тому навіть проміжні файли не синхронізуються з iCloud.

**Помилка `cd: … no such file or directory`?**
Злетіла або неправильно названа закладка — повтори крок 3.

### Як це працює

«Команди» зберігають відео у папку `iCloud Drive/Shortcuts/VideoSplit.nosync` → ffmpeg (усередині a-Shell) ріже його на `part_000.mov`, `part_001.mov`, … потоковим копіюванням (`-c copy -f segment`) → шорткат зберігає частини в галерею по черзі й чистить папку. Технічні деталі та схема всіх дій — у [video-splitter.md](./video-splitter.md).

## ⬇️ Завантажити з YouTube

Відкриваєш відео в YouTube → «Поділитися» → «Завантажити з YouTube». Відео саме з'являється в галереї у форматі, який **CapCut одразу відкриває** — без комп'ютера, без 4K Video Downloader і без Shutter Encoder. Якість — до 1080p (для рілсів і сторіз достатньо).

📱 **Візуальна інструкція:** https://claude.ai/code/artifact/bca5558c-7ba1-48a7-8c9f-d625a0cbf0db

### Встановлення (один раз)

Потрібні **a-Shell** і закладка `~Shortcuts` — так само, як для «Порізати відео» (кроки 1 і 3 вище). Якщо різалка вже стоїть, це вже налаштовано. Далі:

**1. Встанови yt-dlp** — в a-Shell набери:

```
pip install yt-dlp
```

**2. Встанови обидва шорткати** — завантаж на iPhone і відкрий [`Завантажити з YouTube 2.4.shortcut`](./Завантажити%20з%20YouTube%202.4.shortcut) і [`YTSave.shortcut`](./YTSave.shortcut). Помічник `YTSave` не відкривай вручну і не перейменовуй — головний шорткат запускає його сам.

**3. Перший запуск** — поділись коротким відео з YouTube → «Завантажити з YouTube», роздай дозволи.

### Використання

1. YouTube → відео → **«Поділитися»** → **«Завантажити з YouTube»**.
2. Відкриється a-Shell, піде завантаження — просто чекай, нічого не натискай.
3. «Готово! Відео в галереї 🎬» — відкривай CapCut.

### Часті питання

**А можна 4K?**
З телефона — ні. YouTube віддає 4K у кодеку VP9/AV1, який монтажні застосунки не читають без перекодування (саме через це на ПК і потрібен Shutter Encoder). Для соцмереж 1080p достатньо.

**Раптом перестало качати?**
Найчастіше YouTube щось змінив — онови yt-dlp: у a-Shell набери `pip install -U yt-dlp` і пробуй знову.

**Деякі відео не качаються?**
Приватні та вікові (18+) відео недоступні для завантаження — це обмеження YouTube.

### Як це працює

Шорткат відкриває a-Shell (у режимі застосунку, щоб не впертися в ліміт часу розширення), yt-dlp тягне найкраще відео H.264 ≤1080p і склеює зі звуком у mp4, далі рядок `openurl shortcuts://run-shortcut?name=YTSave` запускає помічник, який кладе файл у галерею. Технічні деталі, схема дій і історія рішень — у [yt-downloader.md](./yt-downloader.md).
